# Roodox Encyclopedia / Roodox 项目百科

这个子目录预留给“按主题拆分”的项目百科。目标不是做营销文案，而是把你以后自己修改项目时真正会翻的东西整理成索引化手册。  
This subtree is reserved for a topic-split project encyclopedia focused on maintainers, not marketing.

  Local commands, scripts, and Workbench-exposed interfaces

  Runtime storage, database, certificates, logs, and handoff artifacts
- 源码地图与改动入口  
  Source map and change-entry points
